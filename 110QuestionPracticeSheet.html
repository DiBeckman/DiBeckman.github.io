FLF 110 Question Practice Sheet

There will be a section on asking questions on the final, so we are starting to review it now. Questions are covered in the text in Modules 1.2, 4.5, 5.5. See also: Review Sheet on Asking Questions and Question Jeopardy.

General guidelines:

• Written questions always begin with the question words (Qui… Que… etc)

• Written questions need either est-ce que or inversion. Spoken questions can use intonation.

• Questions, like any other sentence, need a subject and a verb.

Directions: Write another question using the same question word as the model. I've taken models from students' sentences on Exam 3. Choose any topic to write about, but it's always good to practice vocabulary themes from the textbook (childhood, travel, food, etc.) Turn your questions into me to be checked.
