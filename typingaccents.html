Typing Accents

Tex explains how to type accents: http://www.laits.utexas.edu/tex/gr/help.html#acc

You can also find a simple set of instructions for Windows and for Macs on the cengage site.

If nothing else, anyone can access foreign characters simply by using the ASCII code. Basically, every character contained in the computer has a numeric code assigned to it. By manually inputting the code, one can pull out a specific character.

Listed below is a partial list of codes one can use. For additional characters, either experiment on your own or search for a complete listing elsewhere, either on the net or in a reference manual of some sort.

Depress the ALT key and hold it down while you type in the three-digit number listed below. To type the numbers, you must use the numeric keypad on the right side of your keyboard, not the number keys on the
top row.
